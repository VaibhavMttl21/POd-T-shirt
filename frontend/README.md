## Installation

### Prerequisites
- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)

### Setup
1. Clone the repository
   ```bash
   git clone https://github.com/username/project.git
   cd project/frontend
   ```

2. Install dependencies
   ```bash
   npm install
   ```

## Usage
Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
